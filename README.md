# Implementation-of-AES-Encryption-and-Decryption-using-Verilog--Xilinx-ISE-14.7
# Description 
Designed and implemented the AES-128 (Advanced Encryption Standard) algorithm using Verilog HDL and Python for secure data encryption and decryption. The design includes key stages such as SubBytes, ShiftRows, MixColumns, AddRoundKey, and Key Expansion.

# Tools & Technologies :
- Verilog HDL
- Python
- Xilinx ISE 14.7

 # Features :
- Modular implementation of AES stages
- Hardware-based encryption and decryption
- Optimized datapath design
- Functional verification using testbench

#  Results :
- Achieved 30–40% improvement in throughput
- Reduced processing delay by ~25%
- Verified with NIST test vectors (>99.9% accuracy)

#  Applications :
- Secure communication systems
- Cryptographic hardware design
- FPGA-based security implementations
